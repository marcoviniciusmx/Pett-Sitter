## 🏠 About the project

Este projeto foi desenvolvido como uma landing page profissional para a **Beatriz Monteiro - Pet Sitter**, com o objetivo de apresentar os serviços oferecidos, transmitir confiança ao público e facilitar a geração de contato com tutores que buscam um cuidado responsável para seus pets.

A proposta do site é destacar os principais serviços prestados, apresentar a profissional por meio de uma seção institucional, reforçar a proposta de valor da marca e disponibilizar canais de contato de forma acessível e visualmente organizada.

O layout foi pensado para transmitir proximidade, carinho e profissionalismo, com uma navegação simples, visual delicado e boa adaptação para diferentes tamanhos de tela. 

---

## 🧰 Features

- Navegação por âncoras entre seções
- Scroll suave entre áreas da página
- Hero section com slogan e CTA
- Seção de serviços oferecidos
- Blocos visuais com ícones para cada serviço
- Seção “Sobre” com apresentação da profissional
- Botão de solicitação de orçamento
- Footer com contatos e redes sociais
- Efeitos de hover em botões, links e ícones
- Layout responsivo para mobile
- Estrutura simples e objetiva para futura personalização

## 🎨 Layout and design decisions

O projeto utiliza uma identidade visual acolhedora e feminina, com tons rosados, avermelhados e fundos suaves, buscando transmitir afeto, cuidado e confiança — características importantes para um serviço ligado ao bem-estar animal.

Alguns pontos de destaque no layout:

- paleta visual delicada e coerente com o nicho pet
- uso de imagens e ícones para reforçar compreensão rápida
- CTA em destaque para incentivar contato
- organização clara entre apresentação, serviços e contatos
- contraste suficiente para manter boa leitura
- adaptação para mobile sem quebrar a estrutura principal
- interações sutis com hover para enriquecer a experiência do usuário
